***************************
Miscellaneous tools
***************************

abICS comes with a few tools for facilitating typical workflows.

* ``st2abics`` for preparing an abICS input file from a structure file.

* ``abicsRXsepT`` for postprocessing of replica exchange Monte Carlo run.
