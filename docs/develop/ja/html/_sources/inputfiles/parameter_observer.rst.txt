.. highlight:: none

[observer] セクション
-------------------------------

取得する物理量の種類などを指定します.
以下のようなファイルフォーマットをしています.

  :: 

    [observer]
    type = 'default'

入力形式
^^^^^^^^^^^^
``keyword = value`` の形式でキーワードとその値を指定します.
また, #をつけることでコメントを入力することができます(それ以降の文字は無視されます).

キーワード
^^^^^^^^^^

 -  ``type``

    **形式 :** str型 

    **説明 :**
    物理量セットを指定します.
