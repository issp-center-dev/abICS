7. 付属ツール¶

abICSには、典型的なワークフローを容易にするためのツールがいくつか付属しています。

  • 原子構造ファイルからabICS入力ファイルを作成するための st2abics

  • レプリカ交換モンテカルロ計算実行の後処理を行うための abicsRXsepT
